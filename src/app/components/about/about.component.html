About

This is an application that consumes data from fake online REST API JSONPlaceholder and then shows all available doctors, their details and corresponding tasks.
The application uses latest Angular and Karma.

API links:
Doctors: https://jsonplaceholder.typicode.com/users
Tasks: https://jsonplaceholder.typicode.com/todos
Tasks by doctor id: https://jsonplaceholder.typicode.com/users/1/todos
